# 🎓 Previsão de Sucesso Escolar com Inteligência Artificial

Este projeto foi desenvolvido como parte do **Projeto Integrador I** e consiste em um pipeline de dados completo que utiliza Inteligência Artificial para prever se um aluno será aprovado ou reprovado com base em seu histórico e comportamento.

O objetivo principal é identificar alunos em risco de reprovação de forma precoce, permitindo que as instituições de ensino tomem medidas preventivas para evitar a evasão escolar.

## 🧠 Explicação
Imagine que um computador analise o histórico de centenas de alunos, observando detalhes como horas de estudo por semana, ambiente de estudo (barulhento ou silencioso) e nível de escolaridade dos pais. A Inteligência Artificial aprende o "padrão" dos alunos que costumam passar ou rodar de ano. Com isso, quando um novo aluno entra no sistema, o modelo consegue prever com grande precisão se ele precisa de ajuda extra antes mesmo de fazer a prova final.

## 🛠️ Tecnologias e Ferramentas Utilizadas
* **Linguagem:** Python
* **Ambiente de Desenvolvimento:** Google Colab / Jupyter Notebook
* **Manipulação de Dados:** Pandas
* **Visualização de Dados (Gráficos):** Matplotlib e Seaborn
* **Inteligência Artificial (Machine Learning):** Scikit-Learn (Algoritmo de Árvore de Decisão)

## 📊 Etapas do Projeto
1. **Carregamento & Análise:** Leitura da base de dados de estudantes obtida no Kaggle.
2. **Tratamento de Dados:** Limpeza de dados e conversão de textos (como o tipo de ambiente de estudo) em números para que a IA consiga calcular.
3. **Treinamento da IA:** Divisão dos dados em 80% para treino (ensinar a IA) e 20% para teste (avaliar se ela aprendeu direito).
4. **Avaliação:** Extração de métricas oficiais de desempenho (Acurácia, Matriz de Confusão e Relatório de Classificação).
